# LLM Toolkits

A repository of toolkits for various Large Language Models (LLMs).

Manages configurations and resources — including rules, skills, and workflows — for effectively working with LLMs.

## Structure

```
LLM-Toolkits/
└── claude/          # Toolkits for Anthropic Claude
```

## Supported LLMs

- **Claude** (Anthropic) — rules, skills, workflows, CLAUDE.md configurations, etc.

## Contributing

When adding a new LLM or toolkit type, create a directory named after the LLM and organize the relevant files inside it.
